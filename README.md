# Whisper for Sociolinguistics

This is a repository for a code that uses OpenAI’s [Whisper](https://github.com/openai/whisper) to automatically transcribe interviews obtained for the sociolinguistics project EMILS-ISMX (Experiencias Migratorias: Identidad, Lengua y Sociedad. Integración Sociolingüística en México) developed by Ph.D. candidate [Brenda Berenice Larios Loza](https://upol.academia.edu/BrendaBereniceLariosLoza) (Palacký University Olomouc and University of Granada).
